# For Sarah — Click the Button

This is a small pastel-themed web game made as a silly surprise for Sarah. Click the moving button and enjoy playful, romantic, and lightly spicy messages — plus hearts and sparkles.

How to run locally

1. Open `index.html` in your browser (double-click or right-click → Open).

Deploy to GitHub Pages

1. Create a new repo on GitHub and push all files.
2. In the repo settings, enable GitHub Pages and choose the `main` branch root.
3. Your site will be published at `https://<your-username>.github.io/<repo-name>/`.

Files

- `index.html` — main page
- `style.css` — styles and animations
- `script.js` — game interactions and phrases

Customization

- Edit `phrases` array inside `script.js` to tweak messages or order.
- Change colors in `style.css` :root variables to update the palette.

Enjoy, and tell Sarah I said hi 😉
