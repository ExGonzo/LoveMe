## 🚀 How to Deploy (5 Minutes!)

### Step 1: Create GitHub Repo
- Go to [github.com/new](https://github.com/new)
- Name: `LoveMe`, Set to **Public**
- Click **Create**

### Step 2: Upload Files
- Click **Add file** → **Upload files**
- Drag & drop: `index.html`, `game.html`, `css/`, `js/`
- Click **Commit changes**

### Step 3: Enable Pages
- Settings → Pages
- Set Branch to **main** and **/root**
- Click **Save**

### Step 4: Share!
- GitHub shows your live URL
- Send to Sarah: `https://YOUR_USERNAME.github.io/LoveMe/`

**Total time: ~5 minutes⏱️**
