## 🚀 5-Minute Deployment Steps

### ⏱️ (1 minute) Create GitHub Repository

1. Open [github.com/new](https://github.com/new) in your browser
2. Fill in:
   - **Repository name**: `LoveMe`
   - **Description**: "A special website for Sarah 💕"
   - **Visibility**: PUBLIC
3. Click **Create repository**

---

### ⏱️ (2 minutes) Upload Your Files

1. On the new repository page, click **Add file** → **Upload files**
2. **Drag and drop** these items from your computer:
   ```
   ✓ index.html
   ✓ game.html
   ✓ css/ (folder with style.css)
   ✓ js/ (folder with app.js)
   ✓ README.md
   ```
3. At the bottom, click **Commit changes**
4. Wait for the upload to complete (should see green checkmarks)

---

### ⏱️ (1 minute) Enable GitHub Pages

1. On your repository, click **Settings** (top right)
2. Left sidebar → Click **Pages**
3. Under "Build and deployment":
   - **Source**: Select **Deploy from a branch**
   - **Branch**: Select **main** from the dropdown
   - **Folder**: Select **/ (root)**
4. Click **Save**
5. GitHub will start building your site (takes 2-3 minutes)

---

### ⏱️ (1 minute) Get Your Live Link

1. Go back to **Settings** → **Pages**
2. Look for the message:
   ```
   Your site is live at: https://YOUR_USERNAME.github.io/LoveMe/
   ```
3. Copy this URL!

## ⚡ Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "Page not found" | Wait 3-5 minutes, refresh page |
| "404 error" | Make sure files are in root folder (not subfolder) |
| Changes not showing | Clear cache: Ctrl+Shift+Delete, wait 2 min |
| GitHub not responding | Try different browser or wait 5 minutes |
